# Document Collection Guide

Domain: **CSUF Computer Science professor & course reviews**

## Naming convention
- `rmp_lastname_firstname.txt` — one Rate My Professors professor per file
- `reddit_<topic>.txt` — one Reddit thread per file (e.g. `reddit_best_cs_profs.txt`)

## File format
Start every file with a 3-line header so source attribution works during retrieval:

```
SOURCE: Rate My Professors
PROFESSOR/TOPIC: <name or thread title>
URL: <link>

<pasted review text below>
```

## Target: at least 10 files covering different questions
- [ ] 6–7 professor pages (Rate My Professors)
- [ ] 3–4 Reddit threads (course advice, "profs to avoid", general CS dept tips)

## Sources
- Rate My Professors — Cal State Fullerton, filter to Computer Science
- r/CSUF subreddit — search "CPSC professor"
- r/csufcs subreddit (if active)
